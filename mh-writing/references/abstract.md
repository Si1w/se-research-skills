# Abstract

## Format
- Must be a single paragraph.
- Must not contain references.
- Must get to the point quickly.

## Required Elements
Answer these four questions:
1. **What is the problem?**
2. **Why does it matter?**
3. **What is your solution?**
4. **What scientific evidence do you offer that your solution solves the problem?** (Zellerise — include key numbers)

## Writing Process
- Start with each answer as a separate paragraph.
- Focus first on making the answers compelling.
- Once compelling, précis repeatedly until the whole abstract is one paragraph that retains all the semantics.
- Like programming: get it right first, then optimise to make it tight and fast.

## Study Other Abstracts
- Look at abstracts from top venues (ICSE, FSE, ASE, ISSTA, RE, TSE, TOSEM) for format reference.
