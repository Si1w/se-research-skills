# Seven-Dimension Scoring Rubric

A supplementary rubric for the final recommendation in Step 6. The ACM Empirical Standards checklist tells you whether the paper meets methodological requirements; this rubric helps you justify the numerical score (1–10) and the recommendation by decomposing the judgment.

Use this AFTER the checklist evaluation in Step 5. The rubric does not replace the standards-based review — it complements it.

## Dimensions and Weights

| Dimension | Weight | What it measures |
|-----------|--------|------------------|
| Originality | 15% | Genuine novelty of the contribution (method, problem, insight) |
| Methodological Rigor | 25% | Soundness of research design, sampling, statistical analysis |
| Evidence Sufficiency | 20% | Whether the data actually supports the claims |
| Argument Coherence | 15% | Logical flow from problem to conclusion |
| Writing Quality | 10% | Clarity, precision, professional academic prose |
| Literature Integration | 10% | Comprehensiveness and critical positioning against prior work |
| Significance / Impact | 5% | Likely effect on the field or practice |

The weights are calibrated for SE empirical work. For purely theoretical papers, increase Originality and Argument Coherence weights and decrease Evidence Sufficiency.

## Level Descriptors

Each dimension is scored 1 to 5.

### Originality

| Score | Description |
|-------|-------------|
| 5 | Proposes a substantively new method, problem framing, or empirical insight that could redirect the subfield |
| 4 | Clear new insights or novel combinations; fills a specific, well-argued gap |
| 3 | Incremental contribution; reasonable extension of existing work |
| 2 | Heavily overlapping with prior work; the new piece is unclear |
| 1 | Essentially repeats what is already known |

### Methodological Rigor

| Score | Description |
|-------|-------------|
| 5 | Impeccable design; appropriate methods executed without flaws |
| 4 | Sound design; minor room for improvement in execution |
| 3 | Methods acceptable but with design or execution limitations |
| 2 | Significant methodological flaws affecting credibility |
| 1 | Methods fundamentally unsuitable, or contain serious errors |

### Evidence Sufficiency

| Score | Description |
|-------|-------------|
| 5 | Rich, diverse, and persuasive evidence exceeding expectations |
| 4 | Evidence sufficiently supports all major claims |
| 3 | Most claims supported; a few need additional support |
| 2 | Key claims lack sufficient evidence |
| 1 | Serious disconnect between claims and evidence |

### Argument Coherence

| Score | Description |
|-------|-------------|
| 5 | Clear claims, rigorous logic, elegant structure |
| 4 | Smooth argumentation; occasional minor logical leaps |
| 3 | Basically coherent; some inter-section connections unclear |
| 2 | Multiple logical breaks; hard to follow the through-line |
| 1 | Confused argument; core claims cannot be identified |

### Writing Quality

| Score | Description |
|-------|-------------|
| 5 | Precise, fluent academic prose; a model of scholarly writing |
| 4 | Clear; occasional minor imperfections that do not impede understanding |
| 3 | Generally readable; some grammar or word-choice issues |
| 2 | Frequent language issues that affect understanding |
| 1 | Language quality does not meet reviewable standards |

### Literature Integration

| Score | Description |
|-------|-------------|
| 5 | Comprehensive, contemporary, critically integrated; compelling gap argument |
| 4 | Covers major literature with good positioning |
| 3 | Basic coverage with some omissions or shallow integration |
| 2 | Outdated, incomplete, or mere enumeration |
| 1 | Seriously insufficient or off-topic literature treatment |

### Significance / Impact

| Score | Description |
|-------|-------------|
| 5 | Likely to change practice or theoretical direction in the subfield |
| 4 | Clear impact on a specific area of practice or research |
| 3 | Some academic or practical value |
| 2 | Limited scope; mainly academic interest |
| 1 | Significance hard to identify |

## Weighted Total and Decision Mapping

```
Total = 0.15 * Originality
      + 0.25 * Methodological Rigor
      + 0.20 * Evidence Sufficiency
      + 0.15 * Argument Coherence
      + 0.10 * Writing Quality
      + 0.10 * Literature Integration
      + 0.05 * Significance
```

| Weighted total | Recommendation | Note |
|---------------|----------------|------|
| 4.5–5.0 | Accept | Rare. Reserve for outstanding papers. |
| 3.5–4.4 | Minor Revision / Weak Accept | Solid contribution; small fixes needed |
| 2.5–3.4 | Major Revision | Has potential; needs substantial work |
| 1.5–2.4 | Reject (Resubmit possible) | Fundamental issues; topic may have value |
| 1.0–1.4 | Reject | Below publishable standard |

## Override Rules

The weighted total is a guide, not the verdict. Override it when:

- **Methodological Rigor ≤ 2**: Reject regardless of other scores. A paper with broken methods cannot be saved by good writing or a hot topic.
- **Evidence Sufficiency ≤ 2 on a primary claim**: Reject or Major Revision — the contribution is unsupported.
- **Originality = 1**: Reject. A paper with no new contribution does not belong in the venue, even if everything else is competent.
- **A Devil's Advocate CRITICAL finding** (per multi-perspective-lenses.md): cannot be Accept.

Conversely, do NOT inflate the score because the topic is fashionable, the authors are senior, or the writing is polished. The rubric measures the work, not the surface.
