# se-research-skills

A collection of Agent skills for software engineering research paper workflows.

## Skills

| Skill | Description |
|-------|-------------|
| [ese-review](ese-review/) | Review a paper against ACM/SIGSOFT Empirical Standards |
| [mh-writing](mh-writing/) | Review SE paper writing quality based on Mark Harman's guidelines |
| [rp-generate](rp-generate/) | Propose a research project in software engineering or Agent-based systems |
| [git-push](git-push/) | Push the current branch to the remote repository |
